# Exclamation marks series #8: Move all exclamation marks to the end of the sentence
## 7 kyu

Move all exclamation marks to the end of the sentence

### Examples
```
"Hi!"          ---> "Hi!"
"Hi! Hi!"      ---> "Hi Hi!!"
"Hi! Hi! Hi!"  ---> "Hi Hi Hi!!!"
"Hi! !Hi Hi!"  ---> "Hi Hi Hi!!!"
"Hi! Hi!! Hi!" ---> "Hi Hi Hi!!!!"
```
